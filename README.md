# Netty 网关

这是一个基于 Netty 实现的 API 网关项目，或许你可以把它当成一个netty nginx。

目标不是直接做生产可用网关，而是通过逐步实现一个网关，学习 Netty 的核心能力：

- 服务端启动与连接处理
- `ChannelPipeline` 和 `ChannelHandler`
- HTTP 编解码
- 请求转发
- 异步回调与资源管理
- 连接池、超时、限流等网关基础能力
